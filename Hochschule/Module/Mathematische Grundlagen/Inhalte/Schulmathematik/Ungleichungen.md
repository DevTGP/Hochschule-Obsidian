# Beispiel
Geben Sie für die folgende Ungleichung alle Lösungen in $\mathbb{R}$ an: $3* (x-2)^2 \leq 27$

$$
\begin{align*}
&3* (x-2)^2 &\leq 27 && |\div 3\\
&(x-2)^2 &\leq 9 && |\sqrt{}\\
&-3 \leq x-2 &\leq 3 && |+2\\
&-1 \leq x &\leq 5\\
\end{align*}
$$
Somit ist die Lösung der Ungleichung $x \in [-1, 5]$.
