# Spezialregister: Stack Pointer
- Stack = Kellerspeicher 
- Operationen 
	- Push(X) – lege X auf den Stack 
	- Pop() – Nehme oberstes Element vom Stack 
- Arbeitsprinzip: LIFO (Last in First Out) 
- Beispiel: 
	- push(A); 
	- push (B); 
	- push (C); 
	- push (D); 
	- Rx=pop();
