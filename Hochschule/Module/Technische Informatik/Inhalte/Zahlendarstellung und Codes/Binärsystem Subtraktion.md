> [!info] Subtrahieren im Binärsystem 
>>[!error]- Regeln
>>1. **Bitweise Addition** von rechts nach links.
>>2. **Regeln**:
>>    - 0 - 0 = 0
>>    - 1 - 0 = 1
>>    - 1 - 1 = 0
>>    - 0 - 1 = 1 (mit **Borrow** von links, 10 wird zu 1).
>> 3. **Borrow** berücksichtigen.
>> 4. Ergebnis notieren.
 >
>>[!example]- Beispiele
>>>[!note]- $100110 - 001101$
>>> ```math
>>>   100110
>>> - 001101
>>>   110010
>>> --------
>>> = 011001
>>>  ```
^main
